# NumPy Data Explorer

## Project Description
This project demonstrates basic NumPy operations including array creation, indexing, slicing, and mathematical operations.

## Features
- Array creation using NumPy
- Indexing and slicing
- Mathematical operations
- Statistical analysis
- Reshaping arrays
- Save and load NumPy arrays

## Tools Used
- Python
- NumPy

## Outcome
Learned how NumPy is faster and more efficient than Python lists.
